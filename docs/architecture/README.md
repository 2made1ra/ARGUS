# Architecture Diagrams

This directory contains architecture diagram sources for ARGUS.

Render the C4 diagrams with PlantUML:

```bash
plantuml -tsvg c4-container.puml
plantuml -tsvg c4-document-upload.puml
plantuml -tsvg c4-search.puml
plantuml -tsvg c4-document-upload-ru.puml
plantuml -tsvg c4-search-ru.puml
```
